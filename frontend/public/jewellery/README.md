# Jewellery catalogue assets

The catalogue in `src/data/catalogue.ts` references these files, served by Vite
from `frontend/public/jewellery/`:

- gold-necklace.png
- pearl-earrings.png
- diamond-ring.png
- silver-bracelet.png
- classic-watch.png
- round-glasses.png

Drop a square PNG or JPG (transparent or white background works best) for each
name above into this folder. Any product photo of the item is fine. The backend
validates uploads, so files must be real PNG/JPEG/WEBP images.
